Tutorial: Introdução ao HTML

Bem-vindo ao nosso tutorial de introdução ao HTML, a linguagem essencial para construir páginas web.

Elementos Básicos

Os elementos HTML, como <p> para parágrafos e <h1> a <h6> para cabeçalhos, são fundamentais.

Criando Sua Primeira Página

Para começar, crie um arquivo com extensão .html e use um editor de texto simples. Adicione o seguinte código:

<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Minha Primeira Página</title>
</head>
<body>

<h1>Olá, Mundo!</h1>

</body>
</html>
